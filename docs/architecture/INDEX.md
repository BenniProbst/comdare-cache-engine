## Empfohlene Lese-Reihenfolge fuer neue Diplomarbeit-Mitlesende

1. `Architekturentscheidungen_F1_F15.txt` (kanonische Beschluesse)
2. `Domaenenmodell_PRT_ART_v3.txt` (Hauptmodell)
3. `_v4_DELTA.txt` (Korrekturen einarbeiten)
4. `Bausteine_Matrix.txt` (welche Bausteine gibt es)
5. `Flag_System.txt` (Permutations-Identifier)
6. `Begriffsglossar_v6_FINAL.txt` (Begriffe nachschlagen)
7. drawio-Diagramm fuer visuelle Uebersicht
