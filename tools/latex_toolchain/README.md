# Comdare LaTeX-Toolchain (Habich H3, Aufgabe #102)

Dynamische Diagramm-Compile + LaTeX-Build fuer die Diplomarbeit.

## Pipeline

```text
diagrams/*.drawio  ───drawio-cli──→  build/diagrams/*.pdf
        │
        ├── anhang/*.tex  ────────→  build/anhang/*.tex
        │       (generated by tools/latex_anhang from measurements.csv)
        │
        └── main.tex      ─pdflatex→  build/Diplomarbeit.pdf
```

## Voraussetzungen

| Tool | Installation |
|---|---|
| CMake 3.28+        | https://cmake.org (oder Paketmanager) |
| pdflatex / lualatex / xelatex | TeXLive 2024+, MikTeX |
| drawio-cli         | `npm install -g @drawio/drawio-desktop-cli` (optional, Skip wenn fehlt) |
| bibtex             | meistens mit TeXLive zusammen (optional fuer Bibliografie) |

## Aufruf

### Linux/macOS

```bash
./build_thesis.sh \
    --source=/path/to/thesis \
    --output=build/thesis \
    --engine=pdflatex \
    --main=main.tex
```

### Windows

```cmd
build_thesis.bat --source=path\to\thesis --output=build\thesis --engine=pdflatex
```

### Direkt via CMake (Cross-Platform)

```bash
cmake \
    -DCOMDARE_THESIS_SOURCE=/path/to/thesis \
    -DCOMDARE_THESIS_OUTPUT=build/thesis-build \
    -DCOMDARE_LATEX_ENGINE=lualatex \
    -P latex_toolchain.cmake
```

## Konvention fuer thesis/-Verzeichnis

```
thesis/
├── main.tex                      ← Haupt-Dokument
├── refs.bib                      ← Bibliographie (optional)
├── chapters/
│   ├── 01_introduction.tex
│   ├── 02_related_work.tex
│   └── ...
├── diagrams/
│   ├── architecture.drawio       ← wird automatisch zu architecture.pdf
│   ├── permutation_flow.drawio
│   └── ...
└── anhang/
    ├── comdare_table.tex          ← generiert von tools/latex_anhang
    └── ...
```

## Konsistenz zur F-EXTRA-5 NO-PYTHON-Direktive

- `latex_toolchain.cmake` (Standard) — CMake-Backend
- `build_thesis.sh`                    — POSIX-Shell-Frontend
- `build_thesis.bat`                   — Windows-Cmd-Frontend

KEIN Python. Synchron gepflegt zu `tools/permutation_codegen/` Architektur-Pattern.

## Workflow im Comdare-Build-System

```bash
# 1. Permutations-Pipeline (54 DLLs + measurements.csv)
comdare-cache-engine-builder example_configs/ out/ --comdare-root=<repo>

# 2. LaTeX-Anhang-Generation aus measurements.csv
latex_anhang --input=out/measurements.csv \
             --output=thesis/anhang/comdare_table.tex \
             --full --caption="54 COMDARE Permutations" \
             --label="tab:comdare:54perms"

# 3. Thesis-Build (mit Diagramm-Compile + LaTeX)
build_thesis.sh --source=thesis --output=build/thesis-final
```

Resultat: `build/thesis-final/main.pdf` mit allen 54 Permutations-Records +
allen Architektur-Diagrammen.
